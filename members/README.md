# Team Member Images

## Image Requirements

Place your team member photos in this folder with the following specifications:

### File Names:
- `member1.jpg` - CEO
- `member2.jpg` - Chief Engineer
- `member3.jpg` - Project Manager
- `member4.jpg` - Design Director
- `member5.jpg` - Safety Manager
- `member6.jpg` - Quality Control Lead

### Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: Minimum 600x800px (portrait orientation)
- **Aspect Ratio**: 3:4 recommended
- **File Size**: Under 500KB for optimal loading
- **Quality**: High resolution, professional photos

### Tips for Best Results:
1. Use professional headshots or team photos
2. Ensure good lighting and clear backgrounds
3. Consistent style across all photos
4. Optimize images before uploading
5. Use square or portrait orientation

### To Replace Images:
